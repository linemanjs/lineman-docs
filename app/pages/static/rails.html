Lineman + Ruby on Rails

Install Lineman's Rails plugin:

$ npm install --save-dev lineman-rails

This will set up some additional static routes and enable the API proxy. It will also disable the pages task.

Start your lineman development server:

$ lineman run

During development, use Lineman's server:

$ open http://localhost:8000

ruby on rails

ruby on rails

Add this to your Rails app's Gemfile:

gem 'rails-lineman'

Tell the gem where to look for your Lineman app:

config.rails_lineman.lineman_project_location = "my/app"

Alternatively, rails-lineman will look for an environment variable named LINEMAN_PROJECT_LOCATION.

Include your Lineman CSS & JavaScript:

<%= stylesheet_link_tag "lineman/app" %>
<%= javascript_include_tag "lineman/app" %>

Boot your Rails server:

$ bundle exec rails s

When you deploy, Lineman assets will be built & included:

$ rake assets:precompile
