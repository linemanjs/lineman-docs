1. Install [Node.JS](http://www.nodejs.org)
2. Install Lineman globally

   ```bash
   $ sudo npm install -g lineman
   ```
3. Install [Phantom JS](http://www.phantomjs.org) and make sure it's on your _PATH_

   ```bash
   $ sudo npm install -g phantomjs
   ```
